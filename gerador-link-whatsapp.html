Sobre

O gerador de links para o WhatsApp é uma ótima ferramenta para ações de marketing ou relacionamento. Com o link para mensagens personalizadas do WhatsApp, você poderá utilizar em campanhas, redes sociais, email marketing, banners e etc. O bom de encurtar e personalizar links e mensagens do WhatsApp é que funcionará no desktop e no mobile da mesma forma. Faça bom uso da ferramenta encurtadora de WhatsApp.

Instruções

  • Insira o seu número do WhatsApp

  • Escreva a mensagem padrão que será exibida

  • Clique no botão "Gerar link"

  • Copie o link e compartilhe

  • Antes de compartilhar, recomendamos que faça o teste do link

Perguntas de Gerador de Link WhatsApp

Erro: "O número de telefone compartilhado através de url é inválido."

Tente verificar se o número digitado esta correto e que possui o DDI brasileiro (+55).

É possível usar o Gerador de Link WhatsApp para enviar imagens, sons ou vídeos?

Não, o link para WhatsApp é exclusivamente para envio de mensagens de texto defininas previamente para o número selecionado mediante ao clique do link com a mensagem codificada.

O Gerador de Link WhatsApp é Pago?

Não, a ferramenta de Gerador de Link WhatsApp é totalmente gratuíta.

O Gerador de Link WhatsApp salva meus dados?

Não, todos os dados informados são utilizados somente para gerar o link de WhatsApp para o usuário. Os dados não são armazenados em nossos sistemas.
